# Twitter Clone API (Corporate Network Edition)

Асинхронный API-сервис для развертывания внутренней социальной сети. Проект построен на базе **FastAPI** и ориентирован на высокую производительность при обработке пользовательского контента и взаимодействий.

## 🚀 Основные возможности

* **Система твитов**: Публикация текстовых сообщений с поддержкой вложений (изображений).
* **Социальные взаимодействия**: Система лайков, подписок на пользователей и формирование персональной ленты новостей.
* **Асинхронность**: Полностью асинхронный стек (FastAPI + SQLAlchemy Async), обеспечивающий эффективную работу с базой данных без блокировок.
* **Infrastructure-as-Code**: Готовая конфигурация для контейнеризации и оркестрации всей инфраструктуры.

## 🛠 Стек технологий

* **Backend**: Python 3.10+, FastAPI
* **Database**: PostgreSQL
* **ORM**: SQLAlchemy 2.0 (Async)
* **Web Server**: Nginx (используется в качестве Reverse Proxy)
* **Infrastructure**: Docker, Docker Compose

## 📦 Быстрый старт

### 1. Подготовка окружения

Создайте файлы переменных окружения на основе примера. Вам потребуется настроить параметры базы данных и секретные ключи:

* `./main/.env` — параметры приложения и токен.
* `./.env` — настройки подключения к БД (db_login, db_password, db_name).

### 2. Запуск через Docker

Для развертывания всего стека (API + DB + Nginx) выполните команду в корне проекта:

```bash
docker-compose up -d --build

```

После завершения сборки сервис будет доступен по адресу: `http://localhost`.

## 📑 Документация API

Проект поддерживает автоматическую генерацию документации. После запуска сервиса она доступна в форматах:

* **Swagger UI**: `http://localhost:8000/docs` — интерактивное тестирование всех эндпоинтов.
* **Redoc**: `http://localhost:8000/redoc` — структурированное описание API.

### Основные эндпоинты

| Метод | Эндпоинт | Описание |
| --- | --- | --- |
| `GET` | `/api/tweets` | Получение ленты новостей пользователя |
| `POST` | `/api/tweets` | Публикация нового твита |
| `POST` | `/api/medias` | Загрузка изображений для твитов |
| `GET` | `/api/users/me` | Данные текущего профиля |
| `POST` | `/api/users/{id}/follow` | Подписка на пользователя |

## 🧪 Тестирование

Для запуска автоматических тестов используйте команду внутри контейнера или локально:

```bash
pytest tests/

```
